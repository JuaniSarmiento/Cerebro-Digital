# Stack de Herramientas

## Editor / Entorno

- **LazyVim** — distribución de Neovim
- **Tmux** — multiplexor de terminal
- **Zellij** — multiplexor alternativo

## CLI modernos (reemplazos obligatorios)

| En vez de | Uso |
|-----------|-----|
| `cat` | **`bat`** |
| `grep` | **`rg`** (ripgrep) |
| `find` | **`fd`** |
| `sed` | **`sd`** |
| `ls` | **`eza`** |

**Regla**: si la herramienta moderna no está instalada, se instala vía `brew` (o equivalente). No se vuelve a las herramientas viejas.

## Por qué

No es estética. Las herramientas modernas son más rápidas, tienen output legible por humanos por default, sintaxis más sana y mejor integración con el resto del stack. Mantener costumbres viejas por inercia es ruido.
